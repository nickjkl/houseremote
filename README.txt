This repo contains all of the project files for the hackaday prize project HouseRemote.
https://hackaday.io/project/5662-houseremote


The a1284p_automation6 folder contains the firmware for the mcu.

The firstgen folder contains my first draft of files for the web-server side of things and the secondgen is the second draft.

The esp8266msp430v1 contains the code for the MSP430G2553 which communicates to the web server via an esp8266 when the presence of water is detected.
